# Getting Started

Quick start guide for [[projects/WebApp]] development.

## Prerequisites

- [[wiki/Rust]] 1.75+
- PostgreSQL 15+
- Node.js 20+

## Setup

1. Clone the repository
2. Copy `.env.example` to `.env`
3. Run `cargo build`
4. Start the server
